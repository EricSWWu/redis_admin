## redis web client
### 安装

```
python 2.7

pip install -r requirements.txt
python manage.py runserver 0.0.0.0:8000
```

访问浏览器http://127.0.0.1:8000/  
访问用户:admin 密码:admin   
可以使用`python manage.py createsuperuser` 创建用户
